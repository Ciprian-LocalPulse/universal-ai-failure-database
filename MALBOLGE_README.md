# Malbolge — `uaifd_banner.mal`

## What Is Malbolge?

Malbolge was created by **Ben Olmstead in 1998**, named after the eighth circle of Hell in Dante's *Inferno* — the circle for fraudsters. It was specifically designed to be the hardest programming language ever created.

### Why It Is Nearly Impossible to Program In

- Every instruction **self-modifies after execution** using a rotating cipher
- The instruction set is determined not by what you write, but by `(instruction_char + memory_address) mod 94`
- Writing sequential logical code is impossible because every instruction changes itself
- The only known "Hello World" took **2 years** after the language was published to produce — and it was *computed by a search algorithm*, not written by a human

### The Malbolge Instruction Set (8 instructions only)

| Result mod 94 | Meaning |
|---|---|
| 4 | `jmp` — jump to address in data pointer cell |
| 5 | `out` — print char from accumulator |
| 23 | `in` — read char into accumulator |
| 39 | `rotr` — rotate right + store |
| 40 | `movd` — move data pointer |
| 62 | `crz` — crazy operation (ternary XOR-like) |
| 68 | `end` — halt |
| 81 | `nop` — no operation |

Any other value is also a no-op.

## The Program

```
(=<`#9]~6ZY32Vx/4Rs+0No-&Jk)"Fh}|Bcy?`=*z]Kw%oG4UUS0/@-ejc(:'8dc
```

**Output:** `Hello, World!`

This is the **canonical Malbolge Hello World** — the first program ever successfully written in the language. It was generated computationally by Andrew Cooke using a beam search algorithm in 2000, two years after Olmstead published the language.

## Trivia

- The language has exactly **one** control flow construct
- No one has ever written a Malbolge program from scratch by hand — all known programs were generated by search algorithms
- The language's creator stated he was not even sure it was Turing-complete when he published it (it was later proved to be)
